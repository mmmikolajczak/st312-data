# FinArg ECC Argument Unit Classification Dataset (v0)

Dataset module for the ECC argument unit classification task from FinArg-1.

## Dataset ID

`finarg_auc_ecc_official_v0`

## Provenance stance

- Canonical source: local FinArg-1 ECC Argument Classification release copy
- Official split preserved: train / dev / test
- Local packaging is JSON arrays of `[sentence, label]`
- One exact sentence overlap across train/dev is preserved as released and recorded in metadata
